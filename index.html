About Me

Hi, I'm Luke, an information security professional with experience leading security transformations, managing risk, and helping organisations implement international security standards.

This website is a chance to demonstrate some of the projects I've undertaken as I learn and develop in my spare time. I’m particularly interested in building security automation tools and AI-powered systems that improve threat detection and security operations.

I'm currently delivering information security transformation for a leading global pharmacovigilance SaaS organisation, working to unify multiple security systems into a single global security framework.

My background combines cybersecurity, risk management, auditing, and project delivery. Before moving into information security I worked as a project manager delivering complex construction projects while coordinating multi-disciplinary teams and stakeholders. I translated those skills into product management, where I've built features and content into software using AGILE frameworks, taking ideas from the discovery phase through to getting code written, tested and deployed to help solve problems and deliver value.
